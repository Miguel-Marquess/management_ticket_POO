# Sistema de Gerenciamento de Tickets

## Projeto feito em Python que simula um sistema real de Login e venda de tickets para um evento, utilizando Programação Orientada a Objetos e arquitetura modular.

---

## Objetivos 

### O projeto foi desenvolvido na intenção de estudo e pratica da linguagem, focando em:
 - Separação de responsabilidades
 - OOP
 - Estudar como estruturar um sistema escalável 
 - Persistência de dados(Futuramente)

---

## Funcionalidades

 - Cadastro de Clientes
 - Login de usuário
 - Compra de tickets
 - Controle de quantidade de tickets
